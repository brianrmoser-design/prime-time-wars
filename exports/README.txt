Prime Time Wars - Export for Google Sheets

Import these CSVs into Sheets to test and tune rating formulas.

actor_weights.csv   - Rows = show_type, cols = P,C,W,I,ED,DC,SH,CT,TI,BF,LC,CC
writer_weights.csv - One row of weights for W, ED, DC, SH
showrunner_weights.csv - One row of weights for C, W, CC, LC, ED, DC, SH
show_ratings.csv   - Current scores per show (cast, writing, showrunner, base quality)

See FORMULAS.txt for copy-paste Google Sheets formulas and sheet layout.
